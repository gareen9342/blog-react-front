# myblog : frontend

React의 Next.js를 이용해 SSR과 CSR을 적용하여 만든 개인 블로그입니다. 

## 적용한 기술 스택   
  
- React.js
- Next.js 
- Express 
- Node.js
- Sequelize
- Redux & Saga
- Draft.js

## 기능

- [x] 회원가입
- [x] 로그인 
- [x] 비밀번호 찾기 
- [x] 게시물 업로드
- [x] 게시물 수정
- [x] 게시물 삭제
- [x] 게시물 제목 및 내용으로 검색
- [x] 게시물에 댓글 달기
- [x] 게시물에 댓글 수정
- [x] 게시물에 댓글 삭제
- [x] 해쉬태그로 검색
- [x] 게시물에 Like / Unlike
- [x] IP주소를 이용해 게시물 view 수를 카운트
- [x] 방명록 업로드
- [x] 방명록 수정
- [x] 방명록 삭제
- [x] 부분적인 SSR 처리
